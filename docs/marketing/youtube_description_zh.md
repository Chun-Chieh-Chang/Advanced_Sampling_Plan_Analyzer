## 進階抽樣計劃分析器 - 專業品質控制工具

🔬 **用這個全面的抽樣計劃分析工具改變您的品質控制流程！**

### 📊 您將學到什麼：
- **機率分佈分析**：掌握二項式、卜瓦松和超幾何分佈
- **AQL 計劃表查詢**：使用業界標準表格找到最佳抽樣計劃
- **反向抽樣查詢**：從接受/拒收數據計算不良率
- **C=0 計劃優化**：實施零缺陷抽樣策略
- **AQL-LTPD 平衡計劃**：設計平衡生產者與消費者風險的計劃
- **多計劃比較**：並排比較不同的抽樣策略

### 🎯 主要功能：
✅ **互動式 OC 曲線**：即時視覺化操作特性曲線
✅ **AOQ 和 ATI 分析**：分析平均出廠品質和總檢驗數
✅ **風險管理**：計算生產者風險 (α) 和消費者風險 (β)
✅ **匯出功能**：將圖表儲存為 PNG，數據儲存為 CSV
✅ **多種主題**：7 種專業色彩主題，滿足不同偏好
✅ **內建學習**：每個模組都有完整的教學和測驗
